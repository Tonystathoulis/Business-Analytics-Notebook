# Business Analytics Notebook

This repository contains a Jupyter Notebook for a business analytics project focused on hotel booking cancellations.

The notebook explores a hotel booking dataset to identify patterns and factors associated with cancelled reservations. It covers data cleaning, exploratory data analysis, and predictive analysis aimed at helping hotel management better understand cancellation behaviour and reduce related losses.

## Contents
- Analysis of hotel booking demand data
- Investigation of variables influencing cancellation rates
- Data preparation and cleaning steps
- Exploratory visualisations and insights
- Predictive modelling approach for booking cancellation

## Dataset
The analysis uses the public Hotel Booking Demand dataset from Kaggle.

## Notes
This notebook was intended for academic/business analytics study and can be opened and run in Jupyter Notebook or JupyterLab.
